# 大小脸免费初筛 H5 页面

这是一个移动端优先的 H5 自测页面，用于"大小脸免费初筛"项目。

## 功能特点

1. **完整测试流程**：首页 → 12题测试 → 结果页
2. **专业评估逻辑**：基于医学评估标准的评分系统
3. **移动端优化**：触摸友好，响应式设计
4. **本地运行**：无需后端，纯前端实现
5. **模块化设计**：题库、评分规则、结果文案分离配置

## 文件结构

```
h5-face-test/
├── index.html          # 主页面
├── style.css           # 样式文件
├── questions.js        # 题库配置
├── scoring.js          # 评分规则
├── results.js          # 结果文案
├── app.js              # 主应用逻辑
└── README.md           # 使用说明
```

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   ```

## 如何修改

### 1. 修改题目
编辑 `questions.js` 文件：
- 修改 `questions` 数组中的题目内容
- 保持选项的 `value` 值与评分规则一致

### 2. 修改评分规则
编辑 `scoring.js` 文件：
- 修改 `calculateScores` 函数中的计分逻辑
- 修改 `calculateResults` 函数中的判断条件

### 3. 修改结果文案
编辑 `results.js` 文件：
- 修改 `resultTexts` 对象中的文案内容
- 保持文案结构与现有格式一致

### 4. 修改样式
编辑 `style.css` 文件：
- 修改颜色、字体、间距等样式
- 保持移动端友好的设计原则

## 评分规则说明

### 分数类型
- **左右侧分**：判断哪一侧更明显
- **真假分**：判断假性/真性倾向
- **主问题方向分**：5个方向分别计分
- **复杂度分**：判断是否需要进一步评估

### 结果计算
1. **真假倾向**：比较假性分和真性分
2. **主问题方向**：取5个方向中分数最高者
3. **左右侧提示**：比较左侧分和右侧分
4. **复杂度提示**：根据复杂度分和动态控制分判断
